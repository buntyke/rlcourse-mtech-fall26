Policies and Value Functions

MDP Policy
Definition
A policy \(\pi\) is a distribution over actions given states, \[ \pi(a \mid s) = \mathbb{P}[A_t = a \mid S_t = s] \]
  • A policy fully defines the behaviour of an agent
  • MDP policies depend only on the current state (not the history)
  • Policies are stationary (time-independent): \(A_t \sim \pi(\cdot \mid S_t),\ \forall\, t > 0\)
Policy Induces a Markov Reward Process

Given an MDP \(\mathcal{M} = \langle \mathcal{S}, \mathcal{A}, \mathcal{P}, \mathcal{R}, \gamma \rangle\) and a policy \(\pi\):

  • The state sequence \(S_1, S_2, \ldots\) is a Markov process \(\langle \mathcal{S}, \mathcal{P}^\pi \rangle\)
  • The state and reward sequence \(S_1, R_2, S_2, \ldots\) is a Markov reward process \(\langle \mathcal{S}, \mathcal{P}^\pi, \mathcal{R}^\pi, \gamma \rangle\)
Policy Induces a Markov Reward Process

Given an MDP \(\mathcal{M} = \langle \mathcal{S}, \mathcal{A}, \mathcal{P}, \mathcal{R}, \gamma \rangle\) and a policy \(\pi\):

  • The state sequence \(S_1, S_2, \ldots\) is a Markov process \(\langle \mathcal{S}, \mathcal{P}^\pi \rangle\)
  • The state and reward sequence \(S_1, R_2, S_2, \ldots\) is a Markov reward process \(\langle \mathcal{S}, \mathcal{P}^\pi, \mathcal{R}^\pi, \gamma \rangle\)

Where the induced transition and reward functions are:

\[ \mathcal{P}^\pi_{ss'} = \sum_{a \in \mathcal{A}} \pi(a \mid s)\, \mathcal{P}^a_{ss'} \qquad \mathcal{R}^\pi_s = \sum_{a \in \mathcal{A}} \pi(a \mid s)\, \mathcal{R}^a_s \]
State-Value and Action-Value Functions
State-value function \(v_\pi(s)\) — expected return from state \(s\), following \(\pi\): \[ v_\pi(s) = \mathbb{E}_\pi[G_t \mid S_t = s] = \mathbb{E}_\pi\!\left[\sum_{k=0}^{\infty} \gamma^k R_{t+k+1} \;\middle|\; S_t = s\right] \]
Action-value function \(q_\pi(s,a)\) — expected return from \(s\), taking action \(a\), then following \(\pi\): \[ q_\pi(s,a) = \mathbb{E}_\pi[G_t \mid S_t = s, A_t = a] = \mathbb{E}_\pi\!\left[\sum_{k=0}^{\infty} \gamma^k R_{t+k+1} \;\middle|\; S_t = s, A_t = a\right] \]

Bellman Expectation Equations

Bellman Expectation Equation — Derivation

Start from the definition of the state-value function:

\[ v_\pi(s) \doteq \mathbb{E}_\pi[G_t \mid S_t = s] \]
Bellman Expectation Equation — Derivation

Apply the recursive return \(G_t = R_{t+1} + \gamma G_{t+1}\):

\[ \begin{aligned} v_\pi(s) &= \mathbb{E}_\pi[G_t \mid S_t = s] \\[0.4em] &= \mathbb{E}_\pi[R_{t+1} + \gamma G_{t+1} \mid S_t = s] \end{aligned} \]
Bellman Expectation Equation — Derivation

Expand the expectation over actions, next states, and rewards:

\[ \begin{aligned} v_\pi(s) &= \mathbb{E}_\pi[R_{t+1} + \gamma G_{t+1} \mid S_t = s] \\[0.4em] &= \sum_{a} \pi(a|s) \sum_{s'} \sum_{r} p(s',r \mid s,a) \bigl[r + \gamma\, \mathbb{E}_\pi[G_{t+1} \mid S_{t+1} = s']\bigr] \end{aligned} \]
Bellman Expectation Equation — Derivation

Recognise the inner expectation as \(v_\pi(s')\):

\[ \begin{aligned} v_\pi(s) &= \sum_{a} \pi(a|s) \sum_{s'} \sum_{r} p(s',r \mid s,a) \bigl[r + \gamma\, v_\pi(s')\bigr] \end{aligned} \]
\[ v_\pi(s) = \sum_{a} \pi(a|s) \sum_{s',r} p(s',r \mid s,a)\,\bigl[r + \gamma\, v_\pi(s')\bigr] \qquad \textbf{(Eq. 3.14)} \]

The Bellman equation for \(v_\pi\): value of a state = immediate reward + discounted successor value.

Bellman Expectation Equation for \(v_\pi\)

The state value is the average over actions, weighted by the policy:

\[ v_\pi(s) = \sum_{a \in \mathcal{A}} \pi(a|s)\, q_\pi(s,a) \]
Bellman Expectation Equation for \(q_\pi\)

The action value is the immediate reward plus discounted successor value:

\[ q_\pi(s,a) = \mathcal{R}^a_s + \gamma \sum_{s' \in \mathcal{S}} \mathcal{P}^a_{ss'}\, v_\pi(s') \]
Bellman Expectation Equation for \(v_\pi\) — Expanded

Substituting \(q_\pi\) into \(v_\pi\):

\[ v_\pi(s) = \sum_{a \in \mathcal{A}} \pi(a|s) \left( \mathcal{R}^a_s + \gamma \sum_{s' \in \mathcal{S}} \mathcal{P}^a_{ss'}\, v_\pi(s') \right) \]
Bellman Expectation Equation for \(q_\pi\) — Expanded

Substituting \(v_\pi\) into \(q_\pi\):

\[ q_\pi(s,a) = \mathcal{R}^a_s + \gamma \sum_{s' \in \mathcal{S}} \mathcal{P}^a_{ss'} \sum_{a' \in \mathcal{A}} \pi(a'|s')\, q_\pi(s',a') \]
Example: Bellman Expectation Equation in Student MDP
Bellman Expectation Equation: Matrix Form

The Bellman expectation equation can be expressed concisely using the induced MRP:

\[ \mathbf{v}_\pi = \mathcal{R}^\pi + \gamma\, \mathcal{P}^\pi \mathbf{v}_\pi \]

This is a linear system with a direct solution:

\[ \mathbf{v}_\pi = (I - \gamma\, \mathcal{P}^\pi)^{-1} \mathcal{R}^\pi \]

Direct solution is \(\mathcal{O}(n^3)\) for \(n\) states — feasible only for small MDPs.
For large MDPs, iterative methods are needed (covered next lecture).

Optimal Value Functions & Policies

Optimal Value Functions
The optimal state-value function \(v_*(s)\) is the maximum value function over all policies: \[ v_*(s) \doteq \max_{\pi}\, v_\pi(s), \quad \forall\, s \in \mathcal{S} \]
The optimal action-value function \(q_*(s,a)\) is the maximum action-value over all policies: \[ q_*(s,a) \doteq \max_{\pi}\, q_\pi(s,a), \quad \forall\, s \in \mathcal{S},\ a \in \mathcal{A}(s) \]
Optimal Value Functions
\(q_*\) links directly to \(v_*\) (Eq. 3.17): \[ q_*(s,a) = \mathbb{E}\bigl[R_{t+1} + \gamma\, v_*(S_{t+1}) \mid S_t = s,\, A_t = a\bigr] \]

Optimal Policy

Define a partial ordering over policies:

\[ \pi \geq \pi' \quad \text{if}\quad v_\pi(s) \geq v_{\pi'}(s),\ \forall\, s \]
Theorem — For any Markov Decision Process:
  • There exists an optimal policy \(\pi^*\) better than or equal to all other policies
  • All optimal policies achieve the optimal state-value function: \(v_{\pi^*}(s) = v_*(s)\)
  • All optimal policies achieve the optimal action-value function: \(q_{\pi^*}(s,a) = q_*(s,a)\)
Finding an Optimal Policy

An optimal policy can be found by maximising over \(q_*(s,a)\):

\[ \pi_*(a \mid s) = \begin{cases} 1 & \text{if } a = \arg\max_{a' \in \mathcal{A}}\, q_*(s,a') \\ 0 & \text{otherwise} \end{cases} \]
  • There is always a deterministic optimal policy for any MDP
  • If we know \(q_*(s,a)\), we immediately have the optimal policy — no lookahead needed
  • \(q_*\) effectively caches the results of all one-step-ahead searches

Bellman Optimality Equations

Bellman Optimality Equation for \(v_*\)

The value of a state under an optimal policy equals the expected return for the best action:

\[ v_*(s) = \max_{a}\, q_*(s,a) \]

The arc at the agent's choice node represents taking the maximum rather than an average over policy.

Bellman Optimality Equation for \(q_*\)

The optimal action value is the immediate reward plus the discounted optimal successor value:

\[ q_*(s,a) = \mathcal{R}^a_s + \gamma \sum_{s' \in \mathcal{S}} \mathcal{P}^a_{ss'}\, v_*(s') \]
Bellman Optimality Equation for \(v_*\) — Expanded

Substituting \(q_*\) into \(v_*\):

\[ v_*(s) = \max_{a}\left[ \mathcal{R}^a_s + \gamma \sum_{s' \in \mathcal{S}} \mathcal{P}^a_{ss'}\, v_*(s') \right] \]
Bellman Optimality Equation for \(q_*\) — Expanded

Substituting \(v_*\) into \(q_*\):

\[ q_*(s,a) = \mathcal{R}^a_s + \gamma \sum_{s' \in \mathcal{S}} \mathcal{P}^a_{ss'} \max_{a' \in \mathcal{A}}\, q_*(s',a') \]
Example: Bellman Optimality Equation in Student MDP
Solving the Bellman Optimality Equation

The BOE is non-linear — no closed-form solution in general.

Direct solution relies on three assumptions rarely true in practice:

  • The dynamics \(p(s',r \mid s,a)\) are known exactly
  • Sufficient computational resources to complete the calculation
    • Backgammon has \(\sim 10^{20}\) states — thousands of years on today's hardware
  • States satisfy the Markov property
Solving the Bellman Optimality Equation

The BOE is non-linear — no closed-form solution in general.

In practice, we rely on iterative solution methods:

  • Value Iteration — directly iterate on \(v_*\)
  • Policy Iteration — alternate policy evaluation and improvement
  • Q-learning — model-free approximation of \(q_*\)
  • SARSA — on-policy TD control
🖊️ Key insight: Many RL methods can be understood as approximately solving the BOE using experienced transitions in place of known dynamics \(p(s',r \mid s,a)\).
